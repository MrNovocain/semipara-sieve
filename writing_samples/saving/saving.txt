# 一、首先必须确定“在哪个模型里 efficient”

你的 paper 目前是一个 **conditional-mean / conditional-moment paper**，不是完整联合似然模型。最自然的 semiparametric model 是

[
Y_t=m_0(W_{t-1})+\beta_0X_{t-1}+U_t,
]

并假设

[
E(U_t\mid\mathcal G_{t-1})=0,
\qquad
(X_{t-1},W_{t-1})\in\mathcal G_{t-1}.
]

令

[
D_t:=X_{t-1},\qquad W_t^-:=W_{t-1},
]

以及

[
\omega_0(D_t,W_t^-)
:=
E(U_t^2\mid D_t,W_t^-).
]

这里的 nuisance 包括：

[
m_0(\cdot),\quad
\omega_0(\cdot),\quad
\text{以及 }(D_t,W_t^-)\text{ 的联合分布}.
]

我们讨论的是：

> 在只使用条件均值限制
> [
> E{Y_t-m(W_t^-)-\beta D_t\mid D_t,W_t^-}=0
> ]
> 的半参数模型里，(\beta) 的效率界。

这点很重要。因为如果你把

[
D_t=\theta+\rho D_{t-1}+\varepsilon_t
]

和

[
U_t=\phi\varepsilon_t+z_t
]

也作为完整联合似然的一部分，那么一个 fully efficient procedure 可能需要使用 (\varepsilon_t) 和 (\phi) 的信息。那是另一种更强的 efficiency notion，也会迫使你重新使用 AR residuals。

为了保住你现有 paper，最自然的定义是：

[
\boxed{
\text{semiparametric efficiency conditional on the predictor process,}
}
]

即把 (X) 的动态分布视为 nuisance，只用 conditional mean restriction 推断 (\beta)。

这与你现稿“AR(1) 只描述 persistence class，不进入 EL statistic”的设计完全一致。

# 二、所有 orthogonal instruments 长什么样

考虑一个可预测 instrument

[
a_t=a(D_t,W_t^-).
]

相应矩条件为

[
E\left[
a_t{Y_t-m(W_t^-)-\beta D_t}
\right]=0.
]

如果 (m) 沿着方向 (h(W)) 发生扰动：

[
m_\eta(W)=m_0(W)+\eta h(W),
]

那么矩条件关于 nuisance 的一阶导数为

[
-,
E{a_t h(W_t^-)}.
]

要让这个导数对所有 (h) 都为零，必须且只需

[
\boxed{
E(a_t\mid W_t^-)=0.
}
]

这就是 population Neyman orthogonality。

因此，所有合法的 orthogonal instruments 属于

[
\mathcal A
==========

\left{
a(D,W):
E(a\mid W)=0,\quad
0<E(\omega_0a^2)<\infty
\right}.
]

你现在做的

[
w^c=M_Kw
]

正是在样本中强制

[
P'w^c=0.
]

所以从 semiparametric 角度看，你不是在做一个任意的 sieve trick；你是在用有限维 sieve space 逼近条件

[
E(a\mid W)=0.
]

# 三、给定一个 instrument，影响函数和方差是多少

令

[

]

只要

[

]

由矩条件得到的 estimator 具有候选影响函数

[
\phi_a(O_t)
===========

\frac{a_tU_t}{G(a)}.
]

它的方差为

[

====

# E{\phi_a^2}

\frac{E(\omega_0a_t^2)}
{{E(a_tD_t)}^2}.
]

所以 semiparametric efficiency 问题变成：

[
\inf_{a:,E(a\mid W)=0}
\frac{E(\omega_0a^2)}
{{E(aD)}^2}.
]

等价地，最大化 information ratio

[
J(a)
====

\frac{{E(aD)}^2}{E(\omega_0a^2)}.

EL 自身不会自动解决这个问题。EL 能对你给定的 moment 做 self-normalization，但若给它一个低效 instrument，它不会自动恢复被 instrument 丢掉的信息。

[
\boxed{
\text{Wilks validity 和 semiparametric efficiency 是两件事。}
}

[
\widehat Z_{t,\omega}^{\mathrm{eff}}(\beta)

\Omega^{-1}M_{K,\omega}D\

\Omega^{-1}M_{K,\omega}D.

]=0.
]

这意味着

[
E{w(D_t)\mid W_t^-}
\approx
Ew(D_t),
]

所以非恒定的 (W)-projection 在 population 中基本为零。你的现有 lemma 因而证明

[
M_Kw
====

\widetilde w+o_p(1),
]

并把 oracle 保持为 bounded-score 的 centered weight。

但 semiparametric efficiency 想解决的恰恰是：

[
D_t
\text{ 与 }
W_t^-
\text{ 可以相关。}
]

因此 efficient branch 不能继续假设

[
E(D_t\mid W_t^-)=ED_t.
]

它必须显式定义

[
g_0(W)=E(D\mid W)
]

并让

[
M_KD
]

逼近

[
D-g_0(W).
]

所以最准确的说法是：

[
\boxed{
\text{现有 projection algebra 保留；现有 projection-negligible lemma 要推广。}
}
]

旧 lemma：

[
M_Kw-\widetilde w=o_p(1).
]

新 lemma：

[
M_Kw-
\left[
w(D)-E{w(D)\mid W}
\right]
=o_p(1).
]

旧 assumption 只是新框架的特殊情形：

[
E{w(D)\mid W}=Ew(D).
]

# 十五、一个很自然的中间 theorem

你不必立刻从 bounded (w) 跳到 raw (D)。

可以先定义

[
g_w(W)
======

E{w(D)\mid W},
]

[
v_{w,t}
=======

w(D_t)-g_w(W_t^-).
]

若 (w) 是 bounded saturated bounded-score weight，那么 (v_w) 仍然 bounded，并且

[
E(v_w\mid W)=0.
]

样本中继续用

[
w^c=M_Kw.
]

只是 oracle 改为

[
U_tv_{w,t},
]

而不是

[
U_t\widetilde w_t.
]

这样你可以先证明：

[
\frac1{\sqrt T}
\sum_t\widehat Z_t(\beta_0)
===========================

\frac1{\sqrt T}
\sum_tU_tv_{w,t}+o_p(1).
]

这个 theorem 有两个优点：

1. 允许 (X) 与 (W) 相关，spurious-predictability story 变成真的；
2. 仍然保留 bounded weight，所以可能继续覆盖 bounded-score persistence classes。

然后 stationary homoskedastic special case 取

[
w(D)=D,
]

得到

[
v_w=D-E(D\mid W),
]

即 efficient score。

于是整篇 paper 形成很自然的层次：

[
\boxed{
\text{bounded residualized score}
\quad\supset\quad
\text{efficient residualized score as a special case}.
}
]

# 十六、为什么不能马上说“across all persistence regimes efficient”

这里必须谨慎。

你当前 bounded-score theorem 使用 bounded saturated weight，是因为在 integrated 或 mildly explosive regimes 下，(D_t) 自身可能发散，而 bounded (w(D_t)) 仍然保持 score 可控。

但 full efficient score 在 homoskedastic情形是

[
U_t{D_t-E(D_t\mid W_t^-)}.
]

它通常不 bounded。

因此存在一个真实的 trade-off：

[
\boxed{
\text{uniform persistence robustness}
\quad\text{vs.}\quad
\text{full information efficiency}.
}
]

在 stationary regime 中，(\beta) 是标准 root-(T) regular parameter，上面的 semiparametric bound 是清楚的。

但在 local-to-unity、unit-root、mildly explosive triangular arrays 中：

* (\beta) 的自然 local rate 可能不是 (T^{-1/2})；
* information 可能依赖 persistence class；
* limit experiment 可能是 mixed-normal/LAMN；
* efficient information 甚至可能是随机的；
* raw (D_t) score 与 bounded bounded-score score 不再渐近等价。

因此当前最严谨的 claim 应当是：

> semiparametrically efficient in the regular stationary submodel;

而不是：

> semiparametrically efficient uniformly across all persistence regimes.

后者是一个更大的后续 theorem，而不是简单 corollary。

## 1. Oracle efficient score

你的模型是

[

[

E(U_t^2\mid\mathcal H_{t-1}),
]

先考虑平稳的 regular benchmark。取任意 predictable instrument (q_t=q(X_{t-1},W_{t-1}))，moment 为

[
\psi_t(\beta,m;q)
=================

q_t{Y_t-m(W_{t-1})-\beta X_{t-1}}.

对于 nuisance path

[

其 Gateaux derivative 是

[
\left.
\frac{\partial}{\partial\tau}
E{\psi_t(\beta_0,m_\tau;q)}
\right|_{\tau=0}
================

-E{q_ta(W_{t-1})}.
]

因此 score 对未知 (m) 正交，当且仅当

[
E(q_t\mid W_{t-1})=0.
\tag{1}
]

对于满足 (1) 的 instrument，moment estimator 的一阶方差为

[
\operatorname{AV}(q)
====================

\frac{E(\Omega_tq_t^2)}
{{E(X_{t-1}q_t)}^2}.
\tag{2}

现在定义加权投影

[

\arg\min_{h}
E\left[

\right].

[

\frac{X_{t-1}-h_0(W_{t-1})}{\Omega_t}

\right]=0,
]

所以

[
h_0(w)
======

\frac{
E{X_{t-1}/\Omega_t\mid W_{t-1}=w}
}{
E{1/\Omega_t\mid W_{t-1}=w}
}.
\tag{3}
]

令

[
d_t
===

X_{t-1}-h_0(W_{t-1}),
\qquad
q_t^{\mathrm{eff}}
==================

\frac{d_t}{\Omega_t}.
\tag{4}

由 (3)，

所以它确实对 nuisance (m) 正交。

E\left(\frac{d_t^2}{\Omega_t}\right)

\tag{5}
]

[
q_t
\propto
\frac{d_t}{\Omega_t}.

\frac{

\tag{6}

\Omega_t

\phi_t^{\mathrm{eff}}

对于任何满足 aggregate nuisance orthogonality

[
\sum_{t=1}^T
E{q_{t,T}a(W_{t-1})}=0
\quad
\text{for all }a
]

的 instrument sequence，同样有

[
\boxed{
\frac{
\left{\sum_tE(X_{t-1,T}q_{t,T})\right}^2
}{
\sum_tE(\Omega_{t,T}q_{t,T}^2)
}
\le
\mathcal I_T.
}
\tag{18}
]

证明仍然只是 Hilbert-space projection 加 Cauchy–Schwarz。

因此即使 (X) persistent，**optimal-instrument 结论本身没有坏掉**。真正需要新证明的是：这个 optimality bound 是否对应一个 LAMN/semiparametric lower bound，以及 feasible EL 是否达到它。

---

## 6. 你的 Wilks proof 应该怎样升级

现有证明使用

[
T^{-1/2}\sum_t\widehat Z_t
==========================

T^{-1/2}\sum_tU_t\widetilde w_t+o_p(1),
]

以及有限极限

[
T^{-1}\sum_t\widehat Z_t^2
\to_p\eta^2.
]

对于 unbounded efficient score，不应强求方差除以 (T) 后收敛到有限常数。改用 self-normalized 形式。

定义 oracle efficient score

[
Z_{t,T}^\star
=============

U_tq_{t,T}^\star,
\qquad
J_T
===

\sum_{t=1}^T(Z_{t,T}^\star)^2.
]

目标 oracle theorem 是

[
\frac{
\sum_tZ_{t,T}^\star
}{
\sqrt{J_T}
}
\Rightarrow N(0,1),
\tag{19}
]

以及

[
\frac{
\max_t|Z_{t,T}^\star|
}{
\sqrt{J_T}
}
\to_p0.
\tag{20}
]

feasible-oracle equivalence 应写成相对误差：

[
\frac{
\sum_t{\widehat Z_t^{\mathrm{eff}}-Z_{t,T}^\star}
}{
\sqrt{J_T}
}
=o_p(1),
\tag{21}
]

以及

[
\frac{
\sum_t(\widehat Z_t^{\mathrm{eff}})^2-J_T
}{
J_T
}
=o_p(1).
\tag{22}
]

然后你现有的 scalar EL expansion 原样给出

[
\ell_T^{\mathrm{eff}}(\beta_0)
==============================

\frac{
\left{\sum_t\widehat Z_t^{\mathrm{eff}}(\beta_0)\right}^2
}{
\sum_t{\widehat Z_t^{\mathrm{eff}}(\beta_0)}^2
}
+o_p(1)
\Rightarrow\chi_1^2.
\tag{23}
]

所以 proof architecture 仍然是：

[
\text{oracle CLT}
+
\text{feasible-oracle equivalence}
+
\text{EL quadratic expansion}.
]

只是 normalization 从 (T^{-1/2}) 和 (T^{-1}) 攦成由 (J_T) 自适应决定。

广义的 optimally weighted sieve conditional-moment inference同样可以产生标准 chi-square 型结论，但现有结果并不自动覆盖你的 persistent triangular-array setting；这一部分正是需要你新证明的地方。([arXiv][2])

---

## 7. 这篇 paper 最合理的两层 theorem

### Theorem 1：保留你现有结果

> 对任意满足 bounded-score bounded/saturation 条件的 (w)，matched sieve projection 产生 valid EL，并有
> [
> \ell_T^C(\beta_0)\Rightarrow\chi_1^2.
> ]

它强调：

* robustness across persistence classes；
* bounded score；
* standard Wilks；
* 不需要估计 conditional variance。

### Theorem 2：新增 efficiency result

> 取 weighted residualized regressor
> [
> q_t^\star=
> \frac{X_{t-1}-h_0(W_{t-1})}{\Omega_t},
> ]
> 并用 weighted sieve 估计 (h_0,\Omega)。则 efficient EL 保持 Wilks，并达到 conditional-moment semiparametric bound。

再加 local alternatives：

若

[
\frac{\mathcal I_T}{r_T^2}\Rightarrow\mathcal I,
\qquad
\beta_T=\beta_0+\frac{c}{r_T},
]

目标结论是

[
\ell_T^{\mathrm{eff}}(\beta_0)
\Rightarrow
\chi_1^2(c^2\mathcal I),
]

若 (\mathcal I) 随机，则是 mixed noncentral chi-square。对于任何其他 admissible (q)，其 noncentrality 不超过 efficient score，由 (18) 保证。

这才是“semip efficient”真正给 paper 增加的理论深度：不只是方差更小，而是 **最大化 local power**。

## 最终判断

现有工作不会被放弃，但需要诚实地区分：

[
\boxed{
\begin{aligned}
&\text{现有 bounded bounded-score theorem}
&&\longrightarrow
&&\text{robust Wilks branch};\
&\text{新增 residualized-}X/\Omega\text{ score}
&&\longrightarrow
&&\text{efficient branch}.
\end{aligned}
}
]

完全保留的有：

[
m(W)+\beta X,\quad
P_K,\quad
\text{profile nuisance},\quad
\text{matched projection},\quad
\text{EL construction},\quad
\text{quadratic Wilks expansion}.
]

真正新增的是：

[
h_0(W),\quad
\Omega_t,\quad
q_t^{\mathrm{eff}},\quad
\text{information-normalized oracle theorem},\quad
\text{local-power bound}.
]

而你当前的“projection is negligible” lemma 不会成为废稿；它留在 robust theorem。efficient theorem 中，projection 不再是要消灭的误差，而是最核心的有效信息提取步骤。
